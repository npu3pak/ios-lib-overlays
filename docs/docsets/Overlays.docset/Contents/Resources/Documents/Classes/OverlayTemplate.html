View
  • addView(_:)

    Adds the specified view. Please note that the view must have forced height or the intrinsic content size.

    Example:

    let view = UIView()
view.heightAnchor.constraint(equalToConstant: 30).isActive = true
addView(view)

let label = UILabel()
label.text = "example"
addView(label)
